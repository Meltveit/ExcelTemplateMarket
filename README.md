# ExcelTemplates

A professional Excel template marketplace built with React, Express, and TypeScript.

## Features

- 🛍️ Browse and purchase Excel templates
- 💳 Secure payments with Stripe
- 📊 Admin dashboard with sales analytics
- 🔒 Secure file storage and delivery
- 📱 Responsive design

## Tech Stack

- Frontend: React, TypeScript, TailwindCSS
- Backend: Express, Node.js
- Database: PostgreSQL with Drizzle ORM
- Payments: Stripe
- File Storage: Local filesystem with database backup

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/excel-templates.git
cd excel-templates
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Start the development server:
```bash
npm run dev
```
